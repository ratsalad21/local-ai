Store project screenshots for the main [`README.md`](../../README.md) here.

- `bonzo-main-chat.png`
- `bonzo-sidebar-controls.png`
- `bonzo-rag-results.png`
- `bonzo-sidebar-status-sessions.png`
- `bonzo-sidebar-model-kb.png`
- `bonzo-sidebar-explorer-settings.png`

Suggested capture set:

1. Main chat screen with the Bonzo hero and an active response.
2. Sidebar overview showing the full control stack.
3. A retrieval example showing sources or retrieved context in action.

Optional improved sidebar breakdown:

1. `bonzo-sidebar-status-sessions.png`
   Focus on status health and saved sessions.
2. `bonzo-sidebar-model-kb.png`

Tips:

- use real-looking but not sensitive content
- keep the browser width wide enough to show the full layout
- prefer PNG for UI screenshots
- keep images reasonably compressed for GitHub
